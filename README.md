
## 一个云编译UA2F固件的项目

### 食用方法：
##### 用其他设备的话请改Main.config里的上面三行，改成自己所需要的 config
##### 换源，更改Main.yml里的REPO_URL: https://github.com/padavanonly/immortalwrt的地址
##### 如果编译过程中Load custom configuration提示找不到文件，把diy-part2.sh里的mt7621改成相应的内核目录
##### aciton务必使用Main
##### 目前云编译R4A，已测试成功
